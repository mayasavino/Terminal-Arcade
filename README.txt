-------NOTE--------
If you wish to use your own file for custom hangman words, be sure to format
it like hangman.txt, with only a space in between the words. Make sure there
are no symbols or numbers, or it will break.


-------FILES-------
main.py - Arcade
hangman_words.txt - Default file for hangman words


--------RUN---------
1. Navigate to the directory containing the file.
2. Run the file in the terminal with: python arcade.py
3. Play.
